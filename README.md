# Mamba Project

## Overview
This project implements an optimized version of the Mamba model in C.

## Directory Structure
```
mamba_project/
│── include/         # Header files (.h)
│── src/             # Source files (.c)
│── tests/           # Unit tests
│── build/           # Compiled output
│── Makefile         # Makefile for building
│── README.md        # Documentation
```

## Build and Run
```sh
make
./mamba_project
```
